# todolist

Aplikasi sederhana to-do list berbasis command-line.

## Fitur
- Tambah tugas dengan nama, deadline, dan mata pelajaran
- Hapus tugas berdasarkan nama
- Tampilkan semua tugas
- Data disimpan secara otomatis ke file JSON

## Cara Menjalankan
Jalankan `python todo_app.py` di terminal.

## Struktur Data
Setiap tugas memiliki:
- Nama tugas
- Deadline (format YYYY-MM-DD)
- Mata pelajaran
